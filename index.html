🔗 GitHub Auto-Sync
Check Vercel env vars
🔒 GitHub credentials are now stored securely in Vercel environment variables — they never touch your browser. Every "Save All Changes" sends files to your Vercel backend, which pushes to GitHub server-side. Your token is invisible to all visitors and XSS attacks.
✓ Token lives in GITHUB_TOKEN env var on Vercel. To update it, go to your Vercel Dashboard → Settings → Environment Variables.
📄 Resume / CV
🔒 Change Password
Your password is now stored as a bcrypt hash in your Vercel environment variables — never in the browser or in any file. To change it:
  1. Run: node scripts/generate-hash.js yourNewPassword
  2. Copy the hash that is printed
  3. Go to Vercel Dashboard → Your Project → Settings → Environment Variables
  4. Update ADMIN_PASSWORD_HASH with the new hash
  5. Vercel auto-redeploys — done in ~30s
